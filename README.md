# Ruby on Rails 8.0 + Postgres 16.2

This is Ruby on Rails, running on CodeSandbox. Additional to Ruby on Rails, we've configured both Postgres and Redis so you can build a full-stack Rails app on CodeSandbox.

You can check out the [devcontainer configuration](./.devcontainer/devcontainer.json) to see how we configured everything.

You can check out our [task configuration](./.codesandbox/tasks.json) to know what we run on VM start.

# Rails8Base

## Definition
Provides a quick setup of a Ruby on Rails 8 environment for a new project's kick-off.
1. Make sure Rails8Base repository is checked as template repository
2. Create a new repository based on Rails8Base
3. Git clone the new repository
4. Configure and start coding

## Features :
 * /home landing page
 * Devise authentication with sing-up activated
 * Cancan authorisations based on roles
 * Users and roles management
 * rspec and shoulda-matchers for testing
 * postgreSQL based
 * bootstrap or tailwind css
 * swagger API development
